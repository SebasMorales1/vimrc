# My vimrc

## Dependencies
### debian/ubuntu
```bash
sudo apt install fzf bat ripgrep silversearcher-ag perl universal-ctags git curl
```

## Install Vim-Plug
```bash
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

## Move files
```bash
cd vimrc
mv .vimrc ~/
mv .vim ~/
```
